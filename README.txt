STRUCTURE:

  index.html          → Main portfolio page (projects, skills, contact)
  assets/
    css/
      main.css        → All compiled styles (edit this directly)
      noscript.css    → Fallback styles when JS is disabled
    js/
      main.js         → Scroll animations, preload removal, nav effects
      util.js         → Utility helpers
      browser.min.js  → Browser detection
      breakpoints.min.js
      jquery.scrollex.min.js
      jquery.scrolly.min.js
    sass/
      main.scss       → SASS entry point (optional — for advanced use)
      base/           → Reset, typography
      components/     → Buttons, forms, icons
      layout/         → Header, nav, main, footer
      libs/           → Variables, mixins, functions
  images/             → Put your project screenshots here
